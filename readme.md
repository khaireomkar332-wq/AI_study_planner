AI Study Planner

This project helps engineering students generate a personalized study plan for exam preparation.

Inputs:
- Exam date
- Daily study hours
- Subjects
- Hard subjects

The system uses AI-inspired decision logic to prioritize subjects, distribute time intelligently, and generate an optimized study plan.

Tech Stack:
- HTML
- CSS
- JavaScript

No external APIs are required, ensuring reliability and accessibility during exams and low-connectivity environments.
